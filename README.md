# Computational Cognitive Science Training Portfolio

This repository contains a structured progression of practical work in Python-based data analysis, statistical modeling, and computational methods applied to cognitive science and behavioral research.

The goal of this repository is not to present a single polished project, but to document a staged development of core competencies required for computational cognitive psychology and data-driven research.



## Structure

The repository is organized into sequential learning modules:

- Python_Practices  
  Foundational programming concepts and Python syntax.

- NumPy_Practices  
  Numerical computing, vectorization, and linear algebra foundations for data science.

- Pandas_Practices  
  Data wrangling, cleaning, transformation, and structured dataset manipulation.

- Visualization_Practices  
  Scientific visualization, statistical plotting, and publication-quality figures.

- Statistics_Practices  
  Inferential statistics, hypothesis testing, regression, and experimental data analysis.



## Learning Objective

The objective of this project is to develop the computational and statistical foundations necessary for:

- Behavioral and cognitive data analysis
- Experimental psychology research
- Computational modeling foundations
- Reproducible scientific workflows

This work is explicitly oriented toward research preparation rather than application development.



## Methodological Principles

All analyses in this repository follow these principles:

- Reproducibility: all results can be reproduced from raw code
- Transparency: all steps of data processing are documented
- Statistical rigor: assumptions are checked before inference
- Separation of concerns: raw data, processing, and results are clearly separated
- Scientific interpretation: results are interpreted in context, not only computed



## Limitations

This repository represents early-stage development in computational cognitive science.

It does not yet include:

- Bayesian modeling
- Hierarchical models
- Cognitive process modeling (e.g., drift diffusion models)
- Machine learning pipelines
- Large-scale experimental datasets

These will be addressed in future extensions.

## Purpose

This repository is part of a long-term progression toward research competence in computational cognitive psychology, with eventual applications in graduate-level research environments and scientific publication.
