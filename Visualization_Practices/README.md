# Scientific Data Visualization with Python

## Overview

This folder contains a structured set of notebooks designed to teach and demonstrate visualization using Matplotlib and Seaborn mainly. The focus is on visual communication, exploratory data analysis, statistical graphics, publication-quality figures, uncertainty visualization, reproducibility, and visualization design principles relevant to experimental research.

## Folder Structure
- **Practice01_Foundation_Visualization.ipynb** — Introduction to figures, axes, line plots, object-oriented plotting, exploratory versus publication figures, and common visualization pitfalls.
- **Practice02_Relationships_Visualization.ipynb** — Scatter plots, correlation visualization, regression overlays, histograms, KDEs, boxplots, violin plots, bar plots, and distribution diagnostics.
- **Practice03_Matrix_Visualization.ipynb** — Correlation heatmaps, colormap selection, masking strategies, subplot layouts, pairplots, jointplots, catplots, and FacetGrid visualizations.
- **Practice04_PublicationFigures_Visualization.ipynb** — Figure styling, publication-ready formatting, exporting figures, uncertainty visualization, confidence intervals, bootstrap visualization, and regression confidence bands.
- **Practice05_Reproducibility_Visualization.ipynb** — Reusable plotting functions, figure templates, style standardization, and automation of repeated visualization workflows.
- **MiniProject01_Visualization.ipynb** — A visualization project involving dataset generation, missing data handling, exploratory visualization, inferential visualization, and integrated figure panels.

## Key Skills Demonstrated
- Scientific figure design
- Exploratory data visualization
- Publication-quality visualization practices
- Distribution analysis and diagnostic graphics
- Correlation and regression visualization
- Multivariate visualization techniques
- Heatmaps and matrix visualization
- Uncertainty communication
- Bootstrap-based visualization
- Figure layout and composition
- Reproducible visualization workflows
- Automated plotting pipelines
- Research-oriented interpretation of graphics
