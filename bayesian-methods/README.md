\# Bayesian Statistics and Modeling



\## Overview



This folder contains a structured collection of notebooks covering the theoretical foundations and practical application of Bayesian statistics using Python and PyMC. Topics progress from Bayesian probability and parameter estimation to hierarchical models, generalized linear models, model evaluation, Bayesian cognitive modeling, neural modeling, and a complete Bayesian research project. The emphasis is on probabilistic reasoning, uncertainty quantification, reproducible workflows, and research-oriented Bayesian analysis relevant to cognitive science and computational neuroscience.



\## Folder Structure



\* \*\*Practice01\_BayesianFoundations\_Part1.ipynb\*\* — Bayesian probability, Bayes' theorem, conjugate models, posterior updating, parameter estimation, and foundational Bayesian concepts.

\* \*\*Practice02\_BayesianFoundations\_Part2.ipynb\*\* — Bayesian inference, posterior distributions, MCMC, Hamiltonian Monte Carlo, NUTS, diagnostics, and Bayesian workflow fundamentals.

\* \*\*Practice03\_HierarchicalModels.ipynb\*\* — Hierarchical Bayesian models, partial pooling, hyperparameters, multilevel modeling, shrinkage, and hierarchical inference using PyMC.

\* \*\*Practice04\_BayesianRegression.ipynb\*\* — Bayesian linear regression, generalized linear models, prior specification, prior sensitivity analysis, posterior interpretation, and credible intervals.

\* \*\*Practice05\_ModelEvaluation.ipynb\*\* — Posterior predictive checks, model diagnostics, WAIC, LOO cross-validation, Bayesian model comparison, and predictive model assessment.

\* \*\*Practice06\_AdvancedBayesianWorkflow.ipynb\*\* — Variational inference, reproducible Bayesian workflows, model persistence, environment management, and best practices for Bayesian computing.

\* \*\*Practice07\_BayesianCognitiveModel.ipynb\*\* — Bayesian implementation of Signal Detection Theory, simulation, parameter estimation, and interpretation of cognitive processes with uncertainty.

\* \*\*Practice08\_BayesianNeuralModel.ipynb\*\* — Bayesian Linear–Nonlinear–Poisson modeling of neural spike-count data, neural parameter estimation, posterior inference, and uncertainty in neural encoding.

\* \*\*MiniProject01\_BayesianStroop.ipynb\*\* — A complete Bayesian analysis of a simulated Stroop experiment including study design, data simulation, preprocessing, Bayesian regression, posterior inference, posterior predictive checks, publication-quality figures, and interpretation of cognitive effects.
