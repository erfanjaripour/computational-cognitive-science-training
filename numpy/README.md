# NumPy Practices for Behavioral and Experimental Data Analysis

## Overview
This folder contains a structured set of notebooks designed to teach and demonstrate professional use of **NumPy** for scientific data analysis. The focus is on vectorized computation, data manipulation, simulation, reproducibility, and numerical methods relevant to behavioral and experimental research.

## Folder Structure

- **Practice01_nparrayBasics_NumPy.ipynb** — Introduction to `ndarray`, data types, shape, and memory considerations.  
- **Practice02_AccessManipulateData_NumPy.ipynb** — Indexing, slicing, fancy indexing, boolean masking, and views vs copies.  
- **Practice03_Statistics_NumPy.ipynb** — Aggregations, axis logic, and statistical functions.
- **Practice04_SimulationReproducibility_NumPy.ipynb** — Random number generation, simulation of experimental data, seeding, and reproducibility.  
- **Practice05_LinearAlgebra_NumPy.ipynb** — Dot products, matrix multiplication, solving linear systems, rank, and condition number.  
- **Practice06_PerformanceAnalysis_NumPy.ipynb** — Vectorization, memory layout, performance benchmarking, and optimization.  
- **Practice07_RawDataDataframe_NumPy.ipynb** — Preparing arrays for downstream analysis in Pandas, including loading, cleaning, and reshaping.  
- **MiniProject01_Integration_NumPy.ipynb** — Full mini-project integrating simulation, data cleaning, statistical computation, and preparation for analysis pipelines.

## Key Skills Demonstrated
- Efficient array operations and vectorization
- Data simulation and reproducibility
- Data cleaning and preprocessing
- Basic numerical statistics and linear algebra
- Performance optimization
- Preparing data for research-ready pipelines
