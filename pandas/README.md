# Pandas Practices for Behavioral and Experimental Data Analysis

## Overview
This folder contains my Pandas practice notebooks focused on data analysis workflows commonly used in behavioral and experimental research. The notebooks cover the core parts of working with tabular data in Python, including cleaning, filtering, reshaping, grouping, time-series handling, and preparing datasets for analysis.
The goal of these exercises was to build practical familiarity with Pandas through small focused examples and integrated mini-projects rather than isolated syntax demonstrations.

## Folder Structure

- **Practice01_SeriesDataframeBasics_Pandas.ipynb** — Working with Series and DataFrames, basic inspection, indexing, filtering, and descriptive statistics.
- **Practice02_AssignModify_Pandas.ipynb** — Creating, modifying, renaming, and transforming columns while keeping datasets organized and reproducible.
- **Practice03_SortingRanking_Pandas.ipynb** — Sorting values, ranking observations, handling ties, and organizing tabular data.
- **Practice04_GroupbyAggregation_Pandas.ipynb** — Using GroupBy operations for subgroup analysis, aggregation, and summary statistics.
- **Practice05_MergingConcatenating_Pandas.ipynb** — Combining datasets with merges, joins, and concatenation.
- **Practice06_ReshapingPivoting_Pandas.ipynb** — Reshaping data using pivot tables, melting, stacking, and wide/long format transformations.
- **Practice07_ApplyingFunctions_Pandas.ipynb** — Applying custom functions, lambda expressions, and vectorized transformations across datasets.
- **Practice08_StringsCategoricals_Pandas.ipynb** — Cleaning string data, working with text variables, and handling categorical features.
- **Practice09_Timeseries_Pandas.ipynb** — Working with dates and times, resampling, rolling windows, and time-series preprocessing.
- **Practice10_FileIO_Pandas.ipynb** — Reading and writing CSV, Excel, and other common file formats used in data workflows.
- **MiniProject01_Integration_Pandas.ipynb** — A small integration project combining cleaning, transformation, aggregation, reshaping, and analysis steps into a single workflow.

## Topics Practiced
- DataFrame and Series operations
- Data cleaning and preprocessing
- Filtering and conditional selection
- Sorting and ranking
- GroupBy and aggregation
- Merging and reshaping datasets
- String processing and categoricals
- Time-series handling
- File input/output
- Reproducible analysis workflows

## Notes
Most notebooks use small synthetic datasets or simplified experimental-style examples to focus on the Pandas workflow itself rather than domain-specific analysis.
The exercises were written as hands-on practice while learning how to structure cleaner and more reproducible data analysis pipelines in Python.
